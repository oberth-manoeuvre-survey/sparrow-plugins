# SYNOPSIS

Run tests against BashSnippets tools with interactive prompt.

# INSTALL

    $ s6 --install bash-snippets-interactive-test

# USAGE

    $ s6 --plg-run bash-snippets-interactive-test@dir=/path/to/bash-snippets/tools/directory
