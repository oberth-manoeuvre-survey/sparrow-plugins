# SYNOPSIS

Simple wrapper for `speedtest-cli` from [https://github.com/sivel/speedtest-cli](https://github.com/sivel/speedtest-cli).


# INSTALL

    $ s6 --install speedtest-cli

# USAGE

Basic usage:

    $ s6 --plg-run speedtest-cli <args>
