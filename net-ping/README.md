# SYNOPSIS

A simple wrapper around Net::Ping to identify if host is alive.

# INSTALL

    $ s6 --install net-ping

# USAGE

    $ sparrow plg run net-ping --param host=sparrowhub.org

# See also 

[Net::Ping](https://metacpan.org/pod/Net::Ping)
