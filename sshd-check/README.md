# SYNOPSIS

Simple check if your sshd daemon seen at process list.

# INSTALL

    $ s6 --install sshd-check

# Build status

[![Build Status](https://travis-ci.org/melezhik/outth-sshd.svg)](https://travis-ci.org/melezhik/outth-sshd)

# RUN

    $ s6 --plg-run sshd-check
